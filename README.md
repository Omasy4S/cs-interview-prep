# CS Interview Prep

Интерактивный тренажер для подготовки к техническим собеседованиям по Computer Science.

## Что это

Одностраничное веб‑приложение:

- короткая теория с инженерным контекстом;
- рабочие примеры кода (TypeScript/JS);
- мини‑тесты с подробными объяснениями;
- прогресс по разделам сохраняется в `localStorage`.

## Разделы

- **ООП** — инкапсуляция, наследование, полиморфизм, абстракция, `this` в классах
- **Алгоритмы** — бинарный поиск, рекурсия, два указателя, скользящее окно, быстрая сортировка
- **Структуры данных** — стек, очередь, хеш‑таблица, связанный список, дерево
- **SOLID** — SRP, OCP, LSP, ISP, DIP на практических примерах
- **Event Loop** — стек вызовов, очереди задач, `async/await`, `requestAnimationFrame`
- **Основы** — cohesion/coupling, Big‑O, слои архитектуры, компромисс время/память
- **Теория** — библиотека инсайтов: подготовка к собеседованиям, дебаг, product‑thinking, рост инженера и т.п.

## Возможности

- тесты с несколькими вариантами ответа и мгновенной проверкой;
- развёрнутые объяснения «почему так, а не иначе»;
- результат в процентах и мотивационные сообщения;
- адаптивный интерфейс и базовая доступность (ARIA, клавиатурная навигация);
- подсветка синтаксиса через Highlight.js.

## Быстрый старт

1. Клонируйте репозиторий.
2. Откройте `index.html` в браузере.

Никаких зависимостей и сборки не требуется.

## Стек

- HTML5, CSS3
- Vanilla JavaScript (ES6+)
- TypeScript в примерах кода
- Font Awesome, Highlight.js, Google Fonts (Inter)

## Структура

```bash
cs-interview-prep/
├── index.html   # разметка и контейнер для приложения
├── script.js    # конфиг разделов, теория, логика тестов и рендера

```
