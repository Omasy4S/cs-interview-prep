# CS Interview Prep

Интерактивный тренажер для подготовки к техническим собеседованиям.

## Описание

Веб-приложение для изучения и проверки знаний по Computer Science с примерами кода на TypeScript и подробными объяснениями.

## Разделы

- **ООП** - инкапсуляция, наследование, полиморфизм, абстракция
- **Алгоритмы** - бинарный поиск, рекурсия, два указателя, скользящее окно, быстрая сортировка
- **Структуры данных** - стек, очередь, хеш-таблица, связанный список, дерево
- **SOLID** - принципы объектно-ориентированного проектирования (SRP, OCP)
- **Event Loop** - асинхронность в JavaScript
- **Основы** - cohesion, coupling, Big O notation
- **Теория** - концепции языка и архитектуры

## Возможности

- Интерактивные вопросы с примерами кода
- Подсветка синтаксиса кода
- Проверка ответов с мгновенной обратной связью
- Детальные объяснения к каждому вопросу
- Результаты с процентом правильных ответов
- Возможность повторного прохождения тестов
- Навигация между вопросами
- Адаптивный дизайн для мобильных устройств
- Поддержка клавиатурной навигации

## Технологии

- Vanilla JavaScript (ES6+)
- TypeScript для примеров кода
- HTML5
- CSS3 с переменными и градиентами
- Font Awesome для иконок
- Highlight.js для подсветки синтаксиса
- Google Fonts (Inter)

## Установка

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере

Не требуется установка зависимостей или сервер.

## Использование

1. Выберите раздел из вкладок сверху
2. Прочитайте код и ответьте на вопрос
3. Система покажет правильный ответ и объяснение
4. Перейдите к следующему вопросу
5. В конце увидите результаты прохождения

## Структура проекта

```
cs-interview-prep/
├── index.html      # Основная страница
├── script.js       # Логика приложения и данные вопросов
```

## Архитектура

### script.js

**Основные компоненты:**
- `motivationalQuotes` - мотивационные цитаты по результатам
- `appState` - состояние приложения (текущая вкладка, прогресс, счетчики)
- Функции рендеринга интерфейса
- Обработчики событий (клики, навигация)
- Логика проверки ответов и отображения результатов

**Ключевые функции:**
- `buildLayout()` - создание структуры страницы
- `renderTabs()` - отрисовка вкладок разделов
- `renderTabContent()` - загрузка вопросов выбранного раздела
- `handleAnswerClick()` - обработка выбора ответа
- `navigateQuestion()` - навигация между вопросами
- `showResults()` - отображение результатов теста

### styles.css

**Организация:**
- CSS переменные для цветов и теней
- Адаптивная сетка для разных устройств
- Анимации (fadeIn, slideDown, shake, pop)
- Стили для подсветки кода
- Медиа-запросы для мобильных устройств
- Поддержка `prefers-reduced-motion`

## Особенности реализации

- Использование делегирования событий для динамических элементов
- ARIA-атрибуты для доступности (accessibility)
- Показ только одного вопроса за раз с плавными переходами
- Блокировка навигации до ответа на текущий вопрос
- Сохранение состояния прогресса для каждого раздела
- Автоматическая подсветка синтаксиса кода через Highlight.js
