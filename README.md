# Engineering Base | CS Interview Prep

Интерактивная платформа для подготовки к техническим собеседованиям и структурирования инженерных знаний.

## Зачем нужен этот проект

Подготовка к техническим интервью часто превращается в хаотичное решение задач и заучивание определений. Этот проект предлагает другой подход: системное изучение фундаментальных концепций через теорию и практику.

Проект создан, чтобы:
- Структурировать знания по ключевым областям Computer Science.
- Сместить фокус с "заучивания" на понимание паттернов и принципов.
- Предоставить удобный интерфейс для повторения материала перед собеседованиями.
- Объяснить сложные концепции (Event Loop, SOLID, Big O) простым языком на реальных примерах.

## Функциональность

Приложение разделено на модули, охватывающие основные темы технических интервью. Каждый модуль поддерживает два режима:

### 1. Теория
Концентрированная база знаний с объяснением принципов, примерами кода и разбором "почему это важно". Темы подаются через призму инженерного мышления и реальной разработки, а не академических определений.

### 2. Практика
Интерактивные тесты для проверки усвоенного материала. Вопросы проверяют не просто память, а понимание неочевидных моментов (например, поведение `this` в JS, выбор алгоритма под ограничения памяти, нюансы асинхронности).

## Модули обучения

- **ООП**: Инкапсуляция (инварианты), полиморфизм (расширяемость), композиция против наследования.
- **Алгоритмы**: Паттерны решения задач (два указателя, скользящее окно, разделяй и властвуй), оценки сложности.
- **Структуры данных**: Выбор структур под SLA операций (когда нужен массив, а когда связный список или хеш-таблица).
- **LeetCode**: Разбор типовых паттернов задач, чтобы быстро узнавать их условия.
- **Event Loop**: Как работает асинхронность в JS, микрозадачи и макрозадачи, рендеринг.
- **SOLID**: Принципы чистой архитектуры как язык договоренностей в команде.
- **Основы**: Системное мышление, Cohesion vs Coupling, слоистая архитектура.
- **Теория**: Эссе про стратегию подготовки, дебаг и профессиональный рост.
